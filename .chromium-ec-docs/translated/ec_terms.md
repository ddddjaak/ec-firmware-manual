# EC 缩写与技术术语

## 术语表

- **8042 接口**
  向 [AP](#ap) 发送键盘事件并接收 AP 命令的接口。仅支持基于 x86 的 AP。

- **ACCEL — 加速度计**
  测量加速度的传感器，通常为三轴。提供设备方向信息。在 Chromebook 二合一设备中，底座和盖子各有一个加速度计。结合两个加速度计的测量可以精确计算盖子角度，用于在平板模式和笔记本模式之间切换。

- **ACCELGYRO — 加速度计/陀螺仪**
  结合 [加速度计](#accel) 和 [陀螺仪](#gyro) 的组合传感器，通过同时测量线性运动和旋转运动提供更精确的方向信息。

- **ADC — 模数转换器**
  将模拟电压转换为数字读数的传感器。

- **ALS — 环境光传感器**
  测量环境光的传感器。用于自动控制屏幕和键盘背光亮度。

- **AP — 应用处理器**
  板上运行 ChromeOS 的处理器。

- **BAR — 气压计**
  测量大气压力的传感器。

- **BC1.2 — 电池充电规范**
  实现 USB 电池充电规范 1.2 版的设备。完整规范可从 USB 实施者论坛获取。

- **CBI — CrOS 板级信息**
  描述板卡的属性集合，包括板卡版本、SKU、型号名称等字段。

- **CEC — 消费电子控制**
  单线双向总线。

- **DPTF — 动态电源与热框架（Intel）**
  Intel 的平台级电源和热管理方案。

- **EC — 嵌入式控制器**
  用于控制键盘、电池充电、USB 端口切换、传感器管理等功能的 [MCU](#mcu)，将这些任务从 [AP](#ap) 卸载。

- **ectool**
  [AP](#ap) 控制台的命令行工具，用于向 [EC](#ec) 发送命令。

- **EC-3PO**
  当前基于 UART 的控制台的替代方案，将大部分代码从 EC 移到宿主工具中，减少所需的闪存空间。

- **E-Mark — 电子标记线缆**
  参见 USB-C 文档中的相关说明。

- **eSPI — 增强型串行外设接口（Intel）**
  Intel 的同步通信接口，连接 [AP](#ap) 和 [EC](#ec)。支持四线 I/O 模式，时钟频率最高 66 MHz，提供最高 264 Mbps 的带宽。

- **FAFT — 全自动固件测试**
  用于验证 ChromeOS 功能的测试集合和相关基础设施。

- **GMR — 巨磁阻传感器**
  检测磁场的传感器设备。与 [MAG](#mag) 传感器不同，GMR 传感器只检测传感器附近的磁场。在 Chromebook 中用于检测盖子开合。在可翻转 Chromebook 中，当盖子完全打开 360 度时还检测平板模式。

- **GPIO — 通用输入/输出**
  可独立控制和读取的信号。GPIO 用于使能/禁用电源轨、驱动复位信号、接收来自连接到 EC 的设备的中断。

- **GYRO — 陀螺仪**
  测量角动量的传感器，提供设备旋转运动信息。

- **I/O 扩展器**
  提供额外 GPIO 信号的 [I2C](#i2c) 外围设备（8-32 个信号）。通过 I/O 扩展器的 GPIO 使用 EC 中 I2C 控制器的 I2C 寄存器访问进行读写。

- **I2C — 集成电路互连**
  两线制同步通信总线，由时钟信号和双向数据信号组成。I2C 总线通常包含一个控制器设备和一个或多个外围设备。I2C 标准支持 100 KHz 和 400 KHz 的时钟速率。

- **LED — 发光二极管**
  电流通过时发光的半导体器件。

- **LPC — 低引脚数总线**
  [AP](#ap) 和 [EC](#ec) 之间的传统通信总线。运行在 33 MHz，提供 133 Mbps 带宽。已被 [eSPI](#espi) 接口取代。

- **MAG — 磁力计**
  数字罗盘传感器，提供导航方向信息。

- **MCU — 微控制器单元**
  包含 CPU 核心、片上 ROM、片上 RAM 的小型集成电路芯片。还包含多种外设接口，包括 GPIO、I2C 总线、SPI 总线、ADC、PWM 等。

- **MKBP — 矩阵键盘协议**
  用于从 [EC](#ec) 向 [AP](#ap) 通信异步事件的基于消息的协议。事件不限于键盘事件，传感器子系统是主要用户之一。

- **MST — 多流传输**
  DisplayPort 1.2 标准的一部分，用于从单个 DisplayPort 驱动多个独立视频流。EC 代码通常负责使能和禁用 MST 集线器芯片组。

- **OOBM — 带外管理**
  [EC-3PO 协议](#ec-3po) 中的命令，允许在运行时输入命令来改变控制台和解释器的行为。

- **PD — USB 供电**
  参见 USB-C 文档中的相关说明。

- **PMIC — 电源管理集成电路**
  用于开关电源轨的集成电路。

- **PPC — USB 电源路径控制器**
  参见 USB-C 文档中的相关说明。

- **PWM — 脉宽调制**
  通过改变信号占空比来控制其他设备的方法。典型应用是控制风扇转速或背光亮度。

- **Retimer — 信号重定时器**
  可以从高速协议中提取嵌入的时钟信号、完全恢复数据并使用干净的时钟重新传输数据副本的设备。

- **SAFT — 半自动固件测试**
  固件测试套件，已被 [FAFT](#faft) 取代。

- **SHI — SPI 主机接口**
  用于 AP 和 EC 之间通信的 [SPI](#spi) 主机接口。在此配置中，AP 提供 SPI 控制器，EC 提供 SPI 外设。此接口仅用于非 x86 的 AP。基于 x86 的 AP 使用 [eSPI](#espi) 或 [LPC](#lpc) 接口。

- **SPI — 串行外设互连**
  四线制同步通信总线，由 CLK（时钟）、SDO（串行数据输出）、SDI（串行数据输入）和 CS（片选，每个 SPI 外设一个）信号组成。支持超过 100 MHz 的时钟频率。

- **SSMUX — 超高速多路复用器**
  参见 USB-C 文档中的相关说明。

- **SVDM — 结构化厂商自定义消息**
  参见 USB-C 文档中的相关说明。

- **TCPC — USB Type-C 端口控制器**
  参见 USB-C 文档中的相关说明。

- **UART — 通用异步收发器**
  也称为串口。两个设备之间的异步通信通道，具有专用的接收引脚、发送引脚和地线。标准传输速率为 115200 bps。典型用途是为 EC 提供调试控制台。

- **USBA — USB Type-A**
  传统 USB 主机端口。

- **VCONN — 连接器电压**
  参见 USB-C 文档中的相关说明。
