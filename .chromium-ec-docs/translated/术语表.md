# EC 术语与缩写速查

> 覆盖 EC 固件开发中常见的缩写、协议、芯片和工具。每个术语可被其他页面通过锚点引用。

## 系统与架构

**EC** — Embedded Controller，嵌入式控制器。笔记本中独立于主 CPU 的 MCU，负责键盘、电池、热管理、电源时序等。

**AP** — Application Processor，应用处理器。笔记本电脑的主 CPU（x86 或 ARM），运行 ChromeOS / Windows。

**MCU** — Microcontroller Unit，微控制器。EC 所使用的芯片（如 CSCE250X 系列），含 CPU 核、片上 ROM/RAM 和外设接口。

**Zephyr** — Zephyr RTOS，开源实时操作系统（Linux 基金会项目）。Chipsea EC 固件基于 Zephyr 3.7.0 LTS 构建。

**OpenEC** — 开放式 EC 框架。将 EC 固件标准化、模块化的设计理念，降低 ODM 定制成本。

## 通信接口

**eSPI** — Enhanced Serial Peripheral Interface（Intel）。AP 与 EC 之间的同步通信接口，支持四线 I/O，时钟最高 66 MHz，带宽最高 264 Mbps。取代了传统的 LPC 接口。

**LPC** — Low Pin Count bus。AP 与 EC 之间的传统通信总线，33 MHz / 133 Mbps。已被 eSPI 取代。

**SHI** — SPI Host Interface。基于 SPI 的 AP-EC 通信接口，仅用于非 x86 平台（ARM AP）。AP 端为 SPI 控制器，EC 端为 SPI 外设。

**I2C** — Inter-Integrated Circuit。两线制同步串行总线（SCL + SDA），用于 EC 与板上外设（温度传感器、充电 IC、I/O 扩展器）通信。标准速率 100/400 KHz。

**SPI** — Serial Peripheral Interface。四线制同步串行总线（CLK + SDO + SDI + CS），用于高速外设（如 SPI Flash）。时钟可超过 100 MHz。

**UART** — Universal Asynchronous Receiver Transceiver。异步串行通信，典型用于 EC 调试控制台。标准速率 115200 bps。

**SMBus** — System Management Bus。基于 I2C 的管理总线，用于电池、充电器、温度监控等系统管理设备。

**PECI** — Platform Environment Control Interface（Intel）。单线接口，用于读取 CPU 温度、控制风扇等。

---

## USB 与供电

**PD** — USB Power Delivery。USB-C 供电协议，支持最高 100W（PD 3.0）电力传输，以及 Alt Mode（DP、Thunderbolt）。

**TCPC** — USB Type-C Port Controller。USB-C 端口控制器芯片，处理 CC 线检测和 PD 协议物理层。

**PPC** — USB Power Path Controller。电源路径控制器，管理 USB-C VBUS 的开关和限流。

**VCONN** — Connector Voltage。为 USB-C 线缆中的 E-Mark 芯片供电的电源。

**BC1.2** — Battery Charging Specification 1.2。USB 电池充电规范，定义标准充电端口（SDP、CDP、DCP）的识别方式。

**E-Mark** — Electronically Marked Cable。带电子标记的 USB-C 线缆，内含芯片标识线缆能力（电流、速度）。

---

## 电源与热管理

**PMIC** — Power Management IC，电源管理集成电路。用于开关各路电源轨。

**DPTF** — Dynamic Power and Thermal Framework（Intel）。英特尔平台级电源与热管理框架。

**PWM** — Pulse Width Modulation，脉宽调制。通过改变占空比控制风扇转速或背光亮度。

**ADC** — Analog to Digital Converter，模数转换器。将模拟电压（如温度传感器、电流采样）转换为数字值。

---

## 输入与传感器

**GPIO** — General Purpose Input/Output，通用输入输出。可独立控制和读取的引脚，用于使能电源、驱动复位信号、接收中断等。

**KSCAN** — Keyboard Scan，键盘矩阵扫描。通过行列交叉扫描检测按键。

**8042 Interface** — 传统键盘接口，向 x86 AP 发送键盘事件。仅 x86 平台支持。

**MKBP** — Matrix Keyboard Protocol。基于消息的 EC → AP 异步事件协议，不仅传输键盘事件，也承载传感器数据。

**ACCEL** — Accelerometer，加速度计。测量三轴加速度，提供设备方向信息。

**GYRO** — Gyroscope，陀螺仪。测量角动量，提供旋转运动信息。

**ALS** — Ambient Light Sensor，环境光传感器。自动调节屏幕和键盘背光亮度。

**GMR** — Giant Magnetoresistance Sensor。磁阻传感器，检测近距离磁场。Chromebook 中用于检测盖子开合和 360° 翻转（平板模式）。

---

## 安全与启动

**CBI** — CrOS Board Information。板卡信息集合（版本、SKU、型号等），存储在 EEPROM 中。

**OOBM** — Out of Band Management。EC-3PO 协议中的带外管理命令。

**Verified Boot** — 验证启动。确保 EC 固件未被篡改的安全机制。

**Write Protect** — 写保护。硬件/软件机制防止 EC 固件被意外覆盖。

**ERPMC** — Encrypted RPMC（Replay Protection Monotonic Counter）。加密重放保护单调计数器，用于安全存储。

---

## 工具与测试

**ectool** — AP 端命令行工具，用于向 EC 发送 Host Command。

**FAFT** — Fully Automated Firmware Tests。ChromeOS 固件全自动测试套件。

**zmake** — Google 的 Zephyr 构建元工具，用于配置和编译 Zephyr EC 镜像。

**ztest** — Zephyr 原生单元测试框架。

**Renode** — 开源硬件仿真平台，可在 PC 上模拟 EC 运行。

**DTT** — Dynamic Tuning Technology（Intel），动态调优技术。

---

## 其他

**MST** — Multi Stream Transport。DisplayPort 1.2 标准的一部分，从单端口驱动多个独立视频流。

**CEC** — Consumer Electronics Control。单线双向总线，用于消费电子设备控制。

**Retimer** — 信号重定时器。从高速信号中恢复时钟、恢复数据并重发干净信号。

**SSMUX** — Super Speed Multiplexer。超高速 USB 多路复用器。

**SVDM** — Structured Vendor Defined Messages。USB PD 中厂商自定义的结构化消息。
