This document and the linked documents provide a guide for creating new Zephyr EC projects and for modifying/maintaining existing Zephyr EC projects.</p><h2><a class="h" name="How-to-use-this-document" href="#How-to-use-this-document"><span></span></a><a class="h" name="how-to-use-this-document" href="#how-to-use-this-document"><span></span></a>How to use this document</h2><p>The following sections detail a single feature set that needs modification for your board design. Each feature set can be implemented with a reasonably sized change list, and can be worked on independently.</p><p>Each feature includes the following sub-tasks:</p><ul><li><strong>Kconfig Options</strong> - This section details the <code class="code">Kconfig</code> options relevant to the feature. <code class="code">Kconfig</code> options are enabled in one of the [project configuration files].</li><li><strong>Devicetree Nodes</strong> - This section details the devicetree nodes and properties required by the feature. The <a href="https://docs.zephyrproject.org/latest/build/dts/intro.html">Zephyr Introduction to Devicetree</a> provides a conceptual overview of devicetree and how Zephyr uses it.</li><li><strong>Board Specific Code</strong> - When present, this section details any C code that your project must implement.</li><li><strong>Threads</strong> - This section details the threads created by the feature and provides an overview of each thread.</li><li><strong>Testing and Debugging</strong> - This section details strategies for testing the EC feature set and for debugging issues. This section also documents EC console commands related to the feature set.</li><li><strong>Example</strong> - This section walks through a complete example for configuring an EC feature based on existing board implementation.</li></ul><h2><a class="h" name="Adding-a-new-project-to-zmake" href="#Adding-a-new-project-to-zmake"><span></span></a><a class="h" name="adding-a-new-project-to-zmake" href="#adding-a-new-project-to-zmake"><span></span></a>Adding a new project to zmake</h2><p>Refer the <a href="project_config.md">project configuration</a> documentation to add a new board project to zmake.</p><h2><a class="h" name="Configure-EC-Features" href="#Configure-EC-Features"><span></span></a><a class="h" name="configure-ec-features" href="#configure-ec-features"><span></span></a>Configure EC Features</h2><p>The checklist below provides an overview of EC features that you must configure for correct operation of a Chromebook.
