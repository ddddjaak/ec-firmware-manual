title: 功能模块与源码对照
---

快速查找——每个功能模块对应的源码位置。从应用层到驱动层，按需定位。

## 应用层模块（`app/`）

13 个功能模块，每个模块自成一个子目录：

| 模块 | 路径 | 功能 |
|------|------|------|
| 系统基础 | `app/system/` | 系统初始化、版本管理、运行状态 |
| 电源管理 | `app/power/` | 电源状态机、休眠唤醒、电源策略 |
| 电池充放电 | `app/charger/` | 充电曲线、电池保护、电量计算 |
| 热管理 | `app/thermal/` | 温度采集、风扇控制、过热保护 |
| 键盘扫描 | `app/keyboard/` | 键盘矩阵扫描、热键、Fn 组合键 |
| LED 控制 | `app/led/` | 充电指示灯、电源灯、键盘背光 |
| USB Type-C / PD | `app/usbc/` | PD 协议栈、角色切换、Alt Mode |
| 主机通信 | `app/host_if/` | Host Command、ACPI EC Space、SCI/SMI |
| 传感器 | `app/sensors/` | 加速度计、陀螺仪、光感、霍尔 |
| 固件升级 | `app/fw_update/` | OTA、MCUboot、双区切换 |
| 调试接口 | `app/debug/` | Shell 命令、RTT 日志、诊断转储 |
| 安全 | `app/security/` | Secure Boot、写保护、固件签名 |
| 扩展功能 | `app/misc/` | PWM 控制、GPIO 扩展、用户自定义 |

## 驱动层模块（`drivers/`）

18 个外设驱动，按硬件接口分类：

### 通信总线

| 驱动 | 路径 | 挂载总线 |
|------|------|----------|
| I2C 主控制器 | `drivers/i2c/` | I2C |
| SPI 主控制器 | `drivers/spi/` | SPI |
| eSPI 从控制器 | `drivers/espi/` | eSPI |
| PECI 主机 | `drivers/peci/` | PECI |
| UART | `drivers/uart/` | UART |

### 电源与模拟

| 驱动 | 路径 | 功能 |
|------|------|------|
| Fuel Gauge | `drivers/fuel_gauge/` | 电量计（I2C） |
| Charger IC | `drivers/charger/` | 充电芯片（I2C） |
| ADC | `drivers/adc/` | 模数转换 |
| PWM | `drivers/pwm/` | 脉宽调制（风扇、背光） |

### 输入与控制

| 驱动 | 路径 | 功能 |
|------|------|------|
| 键盘矩阵 | `drivers/kbd_matrix/` | 键盘扫描控制器 |
| GPIO 扩展 | `drivers/gpio_expander/` | IO 扩展芯片 |
| 温度传感器 | `drivers/temp_sensor/` | 热敏电阻 / 数字温度计 |

### USB 与存储

| 驱动 | 路径 | 功能 |
|------|------|------|
| TCPC | `drivers/tcpc/` | Type-C 端口控制器 |
| BC1.2 检测 | `drivers/bc12/` | 充电端口识别 |
| SPI Flash | `drivers/spi_flash/` | 外部 Flash 存储 |

### 系统与安全

| 驱动 | 路径 | 功能 |
|------|------|------|
| 看门狗 | `drivers/wdt/` | 硬件看门狗定时器 |
| RTC | `drivers/rtc/` | 实时时钟 |
| CRC | `drivers/crc/` | 硬件 CRC 校验 |
