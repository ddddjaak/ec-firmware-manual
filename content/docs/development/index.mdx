## 如何使用

| 你的角色 | 推荐路径 |
|----------|----------|
| 新加入 EC 开发 | 先看 [架构与配置](/docs/development/architecture/01_layered_architecture) 建立全局认知 |
| 开发新功能 | 直接跳到 [应用开发](/docs/development/appdev/01_new_feature_module) + 参考 [功能模块](/docs/development/modules/01_system_infrastructure) |
| 排查 Bug | 从 [调试与验证](/docs/development/debug/01_logging_and_debug) 入手，用 Shell 命令定位 |
| 需要了解某模块 | [功能模块](/docs/development/modules/01_system_infrastructure) 下按需查阅 |
