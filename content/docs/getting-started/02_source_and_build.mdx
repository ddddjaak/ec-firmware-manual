---
title: zmake 构建工具
---


`zmake` 是 Chromium OS EC 项目的专用构建工具，用于简化 Zephyr 固件的构建流程。Chipsea Zephyr EC 项目当前使用 `west build` + `build.bat` 脚本进行构建，但了解 zmake 有助于理解上游 Chromium EC 的构建体系。

## 基本用法

为单个项目构建 EC 固件：

```bash
zmake build "${PROJECT}"
```

其中 `${PROJECT}` 是项目名称（如 `dedede`、`nissa` 等板级代号）。

## 查看 Kconfig 配置

查看项目的可用 Kconfig 配置选项：

```bash
zmake kconfig "${PROJECT}"
```

这将打开交互式配置界面（类似 `menuconfig`），可浏览和修改项目的 Kconfig 配置。

## 从 Portage 构建

在 Chromium OS chroot 环境中，也可以通过 Portage 系统构建：

```bash
emerge <ec-package>
```

## 与 west 的关系

| 特性 | zmake | west |
|------|-------|------|
| 所属项目 | Chromium OS EC | Zephyr 官方 |
| 构建后端 | CMake/Ninja | CMake/Ninja |
| 项目管理 | 通过项目名自动解析配置 | 需要手动指定 board 和配置 |
| 适用场景 | Chromium OS 通用 EC | Chipsea Zephyr EC / 通用 Zephyr |

> **注意**：Chipsea Zephyr EC 项目使用 `west build` + `build.bat` 构建，不依赖 zmake。此处介绍 zmake 仅供了解上游构建体系参考。
