---
title: 获取源码与编译
---

在环境搭建完成后，下一步是获取 Chipsea Zephyr EC 源码并完成首次编译。整个过程约 **10 分钟**。

<Steps>

<Step>

### 获取工程包

Chipsea Zephyr EC 项目以 SDK 包形式分发，无需从 GitHub 逐个克隆。解压提供的 SDK 到工作目录即可：

```
EC-ZEPHYR
┣ .vscode
┗ .west
  ┗ ecfw-zephyr          # EC 应用代码（app/）
  ┗ ecfwwork
    ┣ modules              # Zephyr 三方模块
    ┗ zephyr_fork          # Zephyr 内核（fork 版）
    ┗ zephyr_modules_csce250x  # 芯海芯片 HAL
```

关键目录说明：

| 目录 | 内容 |
|------|------|
| `ecfw-zephyr/app/` | EC 应用逻辑——13 个功能模块（power、thermal、pd 等） |
| `ecfw-zephyr/drivers/` | EC 设备驱动——18 个外设驱动（fuel gauge、keyboard 等） |
| `ecfw-zephyr/boards/` | 板级支持——`csce250x_evb`、`csce2010_evb` |
| `ecfwwork/zephyr_fork/` | Zephyr 内核 fork——包含芯海芯片适配 |
| `ecfwwork/modules/` | 三方模块——MCUboot、LittleFS 等 |

</Step>

<Step>

### 拉取依赖模块

进入 `ecfw-zephyr` 目录，执行 `west update` 同步所有子模块：

```bash
cd EC-Zephyr/ecfw-zephyr
west update
```

此命令根据 `west.yml` 清单文件拉取所有依赖——包括 Zephyr 内核、HAL 层、MCUboot 等。首次执行需联网下载，后续仅在清单变更时需要重新运行。

> `west update` 仅需在首次搭建或 `west.yml` 变更后执行。日常开发直接使用 `west build` 即可。

</Step>

<Step>

### 首次编译

确保已按 [环境搭建](/zephyr-ec-learning-system/docs/getting-started/01_setup) 完成工具链配置，然后在 `ecfw-zephyr` 路径下执行：

```bash
# 完整构建（pristine build，从头编译）
west build -b csce250x_evb app/ -p
```

**Board Target 选择：**

| 芯片 | Board Target | 说明 |
|------|-------------|------|
| CSCE250X | `csce250x_evb` | Cortex-M33 高性能 EC，主推平台 |
| CSCE2010 | `csce2010_evb` | Cortex-M0+ 轻量 EC |
| CSCE2520 | `csce2520_evb` | 新一代 EC（即将支持） |

编译成功标志——终端输出末尾显示：

```
[finished]  successfully completed
```

产物位于 `build/ec_fw/` 目录：

| 文件 | 用途 |
|------|------|
| `zephyr.elf` | ELF 可执行文件——调试器加载 |
| `zephyr.bin` | 二进制镜像——批量烧录 |
| `zephyr.hex` | Intel HEX 格式——J-Flash 烧录 |
| `zephyr.map` | 内存映射文件——分析 RAM/Flash 布局 |

</Step>

<Step>

### 常用构建变体

```bash
# 增量构建（跳过已编译对象）
west build -b csce250x_evb app/

# 带额外 Kconfig 覆盖
west build -b csce250x_evb app/ -- -DCONFIG_LOG=y

# 使用 release 配置覆盖文件
west build -b csce250x_evb app/ -- -DOVERLAY_CONFIG=release.conf

# 查看 RAM 占用
west build -t ram_report

# 查看 Flash 占用
west build -t rom_report
```

</Step>

<Step>

### 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| `west: command not found` | pip 路径不在 `$PATH` | `pip install west` 后重启终端 |
| CMake 找不到 Zephyr | 未导出 CMake 包 | 执行 `west zephyr-export` |
| `arm-none-eabi-gcc: not found` | 未安装 ARM Toolchain | 安装 Zephyr SDK 0.16.8 或 ARM GNU Toolchain |
| 编译报 python 依赖缺失 | 未安装 requirements | `pip install -r ecfwwork/zephyr_fork/scripts/requirements.txt` |
| `__ZEPHYR_SUPERVISED__` 错误 | 环境变量冲突 | `unset ZEPHYR_TOOLCHAIN_VARIANT` |

</Step>

</Steps>
