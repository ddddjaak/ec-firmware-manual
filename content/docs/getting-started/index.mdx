## 阅读策略

| 你的情况 | 建议 |
|----------|------|
| 第一次接触 Zephyr EC | 按 ①→②→③ 顺序完整阅读，边读边操作 |
| 环境已搭好，只需编译 | 跳过 ①，从 [② 获取源码与编译](/docs/getting-started/02_source_and_build) 开始 |
| 已有固件，只需烧录 | 直接看 [③ 烧录与验证](/docs/getting-started/03_flash_and_verify) |
| 只想快速验证硬件 | 先看 ③ 了解烧录流程，确认板子能跑，再回头补 ①② |
